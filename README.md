# AllTV (WAVVE Live for Android TV)

**AllTV**는 Android TV 환경에서 **WAVVE(웨이브)** 실시간 라이브 채널을 시청할 수 있도록 수정된 앱입니다.

## 📌 프로젝트 소개 및 개발 배경

이 프로젝트는 기존 `ALLTV` 앱에서 WAVVE 라이브 채널이 더 이상 시청 불가능해짐에 따라, 개인적으로 부모님을 위해 수정한 버전입니다.

* 기존에 `Allive TV` 등을 통해 수정해 주시던 분이 계셨으나, 최근 WAVVE 업데이트로 인해 다시 채널이 보이지 않게 되었습니다.
* 업데이트가 언제 될지 기약이 없어, 직접 수정하여 부모님 댁 TV에 설치해 드리고 **자동 업데이트**를 받을 수 있도록 하기 위해 이 리포지토리를 생성했습니다.
* 저는 안드로이드 앱 개발자가 아니며(타 언어 개발자), 기존 소스의 구조나 안드로이드 앱의 동작 방식을 깊이 알지 못한 상태에서 오로지 **WAVVE 시청**만을 목적으로 수정하였습니다.

## 🛠 수정 및 변경 사항

* **WAVVE 라이브 채널 복구:** 기존 소스에서 로그인 및 채널 목록 로딩 방식을 수정하여 다시 동작하도록 만들었습니다.
* **개발 환경 최신화:** 기존 프로젝트 코드를 가져와 현재 빌드 환경에서 돌아가도록 버전을 올리고 수정했습니다.
* **옥수수(Oksusu) 제거:** 서비스가 종료된 옥수수 관련 기능은 코드에서 제거했습니다.
* **자동 업데이트 지원:** 앱 내에서 새 버전을 감지하고 업데이트할 수 있도록 구성했습니다.

## ⚠️ 주의사항 (Known Issues)

이 앱은 개인적인 목적으로 급하게 수정되었으며, 다음과 같은 사항을 유의해 주시기 바랍니다.

1.  **티빙(Tving) 미지원:** 저는 WAVVE 계정만 보유하고 있어 티빙 쪽 소스는 전혀 건드리지 않았습니다. 따라서 **티빙 기능은 동작하지 않습니다.**
2.  **잠재적 버그:** 안드로이드 전문 개발자가 아니므로 예상치 못한 버그가 존재할 수 있습니다.
3.  **소스 코드 비공개:** 보안 문제 및 계정 정보 유출 우려, 그리고 allive TV 를 만들어주시던 분의 선례를 따라 **소스 코드는 공개하지 않고 릴리즈(APK) 파일만 배포**합니다.
