# alltv
alltv releases

Alltv 는 Android TV 에서 WAVVE live 채널을 시청할 수 있는 AndroidTV 앱입니다.
이 프로젝트는 ALLTV 에서 더 이상 wavve live 채널을 시청 할 수 없어서 개인적으로 부모님을 위해 개발 해서 자동으로 업데이트 받을 수 있도록 레포지토리를 만들어 놓은 곳입니다. 저는 wavve만 계정이 있어서 기존 소스에 wavve쪽만 수정하여 시청 할 수 있도록 하였습니다.
기존에 alltv에서 안쪽 소스가 뭘 쓰고 있는지 저장은 어디에 되는지 안드로이드 앱은 어떤식으로 되는지 아무것도 모르는 일반 다른 언어 개발자 입니다.
원래 다른 분이 allive tv로 수정 해주시고 계셨는데 이번 wavve에서 업데이트되고 채널이 안보여서 언제 업데이트가 될지 몰라 개인적으로 수정 후 업로드만 해놓습니다. 보안 및 계정 정보로 인해 소스를 공개 안하시던데 먼가 기존 코드에도 동일한 문제가 있을까 하여 코드는 공개하지 않도록 하겠습니다.
기존 프로젝트에서 코드를 가져와서 어떻게든 개발환경에 맞도록 버전 올리고 수정하고 돌아가도록 만든 뒤에 로그인 부분과 라이브 채널 부분만 고쳐서 동작 하도록 만들었습니다.
어마어마한 버그가 있을 수도 있고 티빙 쪽은 손도 대지 않아서 동작하지 않을 것입니다. 그리고 옥수수는 제거 했습니다.

혹시나 이 레포지토리에 문제가 있는 경우는 알려주세요.
